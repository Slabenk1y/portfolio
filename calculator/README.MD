# Калькулятор

Простой консольный калькулятор на Go.

## Возможности

- ✅ Сложение (+)
- ✅ Вычитание (-)
- ✅ Умножение (*)
- ✅ Деление (/)
- ✅ Проверка деления на ноль

## Как запустить

```bash
go run main.go
```

## Пример
=== КАЛЬКУЛЯТОР SLABENK1Y ===

Первое число: 7
Операция (+,-,*,/): *
Второе число: 27

Результат: 7 * 27 = 189
